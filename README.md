# LibraBrowser
Code for LibraBrowser.io

## Installation
1. Install Libra per official instructions
2. make sure the CLIENT_PATH variable is correct
3. pip3 install grpcio grpcio-tools

## Running the project
At the root project folder execute the command:
> python3 Browser.py
